# cat-gatekeeper-desktop-test

Repository: clocktest

A test-only desktop prototype inspired by break reminder workflows.

## Important notice

This repository intentionally uses **test** in its name and is not intended for production distribution.

Original Cat Gatekeeper project includes assets with separate license terms. This prototype does **not** bundle those assets.

## Run

```bash
npm install
npm start
```

## Features in this quick prototype

- Configurable interval and break duration
- Start / Pause / Skip controls
- Desktop break window with countdown
- Local settings persistence

## Stack

- Electron
- Vanilla HTML/CSS/JS
